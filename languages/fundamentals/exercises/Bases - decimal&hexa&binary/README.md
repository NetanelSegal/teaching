# Number Bases Exercises

This folder contains exercises for converting between different number bases: Decimal, Binary, and Hexadecimal.

## Binary Conversions
1. **Decimal to Binary**:
   - Convert 1, 2, 4, 8 to binary.
   - Convert 56, 256, 512, 1024, 893, 278, 81 to binary.
2. **Binary to Decimal**:
   - Convert `1101` to decimal.
   - Convert `1010101010` to decimal.
   - Convert `100000110` to decimal.
   - Convert `100110100110` to decimal.

## Hexadecimal Conversions
3. **Hexadecimal to Decimal**:
   - Convert `3E8`, `1A7F`, `7B3A`, `2F5B9`, `ABCD`, `FFFF` to decimal.
4. **Decimal to Hexadecimal**:
   - Convert 1024, 2048, 65535, 1048576, 16777215, 4294967295 to hexadecimal.
5. **Hexadecimal to Binary**:
   - Convert `1F4`, `3A9C`, `7FFF`, `ABCDEF`, `123456`, `FFFFFFFF` to binary.
6. **Binary to Hexadecimal**:
   - Convert various long binary strings (e.g., `1010101010101010`, `1111111111111111`) to hexadecimal.

## Negative Numbers and Two's Complement
7. **Find Two's Complement**:
   - For binary strings: `0101010101`, `01100110`, `011111001`, `01001100`.
8. **Negative Decimal to Binary**:
   - Convert -7, -38, -127, -57 to binary (using appropriate bit-width).

## Binary Arithmetic
9. **Addition and Subtraction**:
   - `10010011 + 00011001`
   - `1001 + 0011`
   - `1111 - 0101`
   - `11000011 - 00001011`
