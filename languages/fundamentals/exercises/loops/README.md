# Loop Exercises

This folder contains exercises for working with loops (`for`, `while`, `do-while`) in C and Java.

## Basic Loops
1. **Sports Form**: Input 15 characters (representing '1', '2', or 'X'). Count how many times 'X' appears.
2. **Sum in Range**: Input two integers and display the sum of all integers between them (inclusive).
3. **Homogeneous Group**: Input $N$ ages. A group is "homogeneous" if the difference between the oldest and youngest is at most 3 years. Otherwise, it is "heterogeneous".
4. **Election**: In a class of 41 students, count votes for two candidates (1 or 2). Display the winner.

## Intermediate Logic
5. **Time Formatter**: Input a number of seconds (0-86400) and display it in `HH:MM:SS` format.
6. **Consecutive Sum**: Input $N, M$, and then $N$ numbers. Check if there exist any $M$ consecutive numbers in the input whose sum is greater than $M$.
7. **Chess Threats**:
   - Given the coordinates of a Rook and a Bishop, determine if they threaten a specific square $(row, col)$.
   - A square is threatened by a Rook if they share a row or column.
   - A square is threatened by a Bishop if they share a diagonal.

## Sequences and Math
8. **Geometric Sequence**: Input the first element $a_1$, the ratio $q$, and the number of elements $n$. Display the first $n$ elements of the geometric sequence.
9. **Multiples of 3 and 5**: Find the sum of all natural numbers below 1000 that are multiples of 3 or 5.
10. **Digit Sum Pattern**: For a given $a \in [1, 18]$, find all 2-digit numbers where the sum of their digits equals $a$.
11. **Lucky Fractions**: Find all pairs of 2-digit numbers $AB/BC$ such that "cancelling" the digit $B$ results in a fraction $A/C$ equal to the original fraction (e.g., $16/64 = 1/4$).

## Nested Loops (Java)
12. **Multiplication Table**: Print a $10 \times 10$ multiplication table.
13. **Star Patterns**:
    - Right-angled triangle of stars.
    - Isosceles triangle.
    - Diamond shape.
14. **Prime Numbers**: Print all prime numbers between 1 and 100.
