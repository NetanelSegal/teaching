# Array Sorting Algorithms 📉

## Goal

Understand and implement fundamental sorting algorithms.

## Instructions

Implement the following sorting algorithms in your chosen language:

1. **Bubble Sort**: Repeatedly step through the list, compare adjacent elements and swap them if they are in the wrong order.
2. **Selection Sort**: Find the minimum element in the unsorted portion of the array and swap it with the first unsorted element.
3. **Insertion Sort**: Build the final sorted array one item at a time by inserting each element into its proper place.
4. **Merge Sort** (Advanced): Divide the array into halves, sort them recursively, and merge them back together.
