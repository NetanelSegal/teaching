# Boolean and Conditional Exercises

This folder contains exercises for working with boolean logic and `if-else` statements.

## Basic Conditionals
1. **Sign Check**: Input a number and print whether it is positive, negative, or zero.
2. **Parity Check**: Input a number and print whether it is even or odd.
3. **Character Classification**: Input a character and determine if it is a lowercase letter, uppercase letter, digit, or other symbol.

## Intermediate Logic
4. **Ordered Numbers**: A 2-digit number is "ordered" if its tens digit is greater than its units digit. Input a 2-digit number and check if it is ordered.
5. **3-Digit Palindrome**: Input a positive 3-digit number and determine if it is a palindrome (reads the same forward and backward, e.g., 787, 424).
6. **Shuttle Capacity**: A class of students is going on a trip to the moon. Each shuttle has a fixed number of seats. Given the total number of students and the capacity of one shuttle, calculate the minimum number of shuttles required so that every student has a seat.
