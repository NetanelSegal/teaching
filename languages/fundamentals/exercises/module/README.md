# Modulo Operations (%) ➗

## Goal
Master the use of the modulo operator for cycles, digit extraction, and divisibility.

## Instructions
Implement the following logic:

1.  **Even/Odd Check**: Check if a number is even or odd using modulo.
2.  **Digit Extraction**: Given a 3-digit number, extract and print the hundreds, tens, and units digits individually.
3.  **Leap Year**: Check if a given year is a leap year (divisible by 4 and not 100, or divisible by 400).
4.  **Clock Math**: Given a current hour (0-23) and a number of hours to add, print the new hour on a 24-hour clock. (e.g., 22 + 5 hours = 3).
