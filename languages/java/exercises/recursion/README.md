# Recursion Drills 🔄

## Goal
Master the concept of Recursion by solving a series of increasingly complex problems in your chosen language without using traditional loops (like `for` or `while`).

## Prerequisites
- Language Syntax Fundamentals
- Understanding of Methods and Return Types
- Conceptual understanding of Call Stacks and Base Cases

## Instructions
Open the `starter` folder. You will find a text file `recursion functions to implement.txt` containing method signatures. Your task is to implement each method using **only recursion**.

### The Challenges:
1. **Printing**:
   - `printAscendingOrder(int n)`: Print numbers from 1 to `n`.
   - `printDescendingOrder(int n)`: Print numbers from `n` to 1.
2. **Number Analysis**:
   - `numOfDigits(int n)`: Return how many digits are in `n`.
   - `sumOfDigits(int n)`: Return the sum of the digits in `n`.
   - `biggestDigit(int n)`: Return the largest single digit inside `n`.
   - `AreAllDigitsEven(int n)`: Return `true` if every digit in `n` is an even number.
   - `isDivisor(int num, int divisor)`: Check if `divisor` evenly divides `num`.
3. **String Manipulation**:
   - `seekString(String str, char letter)`: Return `true` if `letter` exists in `str`.
   - `isPalindromeRec(String str)`: Return `true` if the string is exactly the same forwards and backwards.

## Structure
*   `/starter` - Your workspace. Contains the method signatures you need to implement.
*   `/solution` - Contains the working, tested solutions. No peeking until you've tried!
