# OOP in JavaScript 🏗️

## Hook 🪝
If you're building a game, you might have hundreds of "Enemies." Instead of creating each one from scratch, you can use a blueprint (a **Class**) to define what every enemy can do. This is **Object-Oriented Programming (OOP)**—a way to organize your code around objects and data.

## Theory 📚
Modern JavaScript uses `class` syntax to create reusable blueprints for objects.

### Key Concepts:
- **`class` and `constructor`**: Defining the blueprint.
- **Methods**: Functions that belong to an object.
- **`this`**: Accessing the current object's properties.
- **Inheritance**: Creating a "SuperEnemy" based on the "Enemy" class.

Refer to `theory.js` for structured examples.
