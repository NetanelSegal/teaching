# JavaScript Fundamentals (Lessons 1-3)

This folder contains exercises focusing on the core foundations of JavaScript.

## Topics Covered:
1.  **Variables & Scope**: `let`, `const`, naming conventions, and the swap algorithm.
2.  **Data Types**: Deep dive into all 8 types (Number, BigInt, String, Boolean, null, undefined, Symbol, Object).
3.  **Basic Operators**: Mathematical operations, increment/decrement, and modulo.
4.  **Type Conversion**: Understanding how JS converts data types automatically and manually.

## Files:
-   `exercises.js`: The practice sheet with 7 multi-part challenges.
-   `solutions/solution.js`: Reference implementations with detailed explanations.

## How to use:
1.  Open `exercises.js` in your editor.
2.  Write your solutions below each comment block.
3.  Run the file using `node exercises.js` in your terminal to check the output.
4.  Compare your results with the `solutions/solution.js` file if you get stuck.
