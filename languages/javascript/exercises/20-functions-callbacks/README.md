# Exercises 20: Functions & Callbacks

## Goal
Master the use of functions as first-class citizens in JavaScript. Understand the differences between definitions, arrows, and callbacks.

## Topics Covered
1. **Default Parameters**: Handling missing arguments gracefully.
2. **Hoisting**: Difference between Declarations and Expressions.
3. **Arrow Functions**: Concise syntax for simple logic.
4. **Higher-Order Functions**: Functions that accept other functions.
5. **Callbacks**: Passing logic as a value.
