# Lesson 01: Algorithms & Math Logic

This folder contains exercises focused on pure logic and mathematical calculations using JavaScript. These tasks will help you master:
- Mathematical operators (+, -, *, /, %, **)
- Comparison and Logical operators
- Converting real-world formulas into code
- Working with numbers and basic logic

## Exercises
- **Basic Arithmetic**: Implementing formulas like Area, Average, and BMI.
- **Modulo Logic**: Identifying even/odd numbers and extracting time units.
- **Logical Flow**: Using conditional logic to classify values.

## How to use
Open `exercises.js` and write your solutions under each task comment. You can run your code using a browser console or Node.js.
