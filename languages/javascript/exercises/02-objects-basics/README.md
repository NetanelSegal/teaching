# Lesson 02: Objects & User Inputs

This folder explores how JavaScript works with more complex data types like Objects and how it interacts with the user via built-in browser functions.

## Key Concepts
- **Objects**: Creation, access (dot vs bracket notation), and manipulation.
- **User Interaction**: Using `alert()`, `prompt()`, and `confirm()`.
- **Parsing**: Converting string inputs to numbers (`parseInt`, `Number`).
- **Template Literals**: Creating dynamic strings.

## Exercises
- **The "User Profile"**: Dynamic object creation from user prompts.
- **Library System**: Managing nested object data.
- **Calculator Prototype**: Interacting with inputs and performing logic.

## How to use
Open `starter/exercises.js` and write your solutions under each task comment.
