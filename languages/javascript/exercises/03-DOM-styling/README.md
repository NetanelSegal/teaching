# Lesson 03: DOM Selection & Styling

This exercise marks the transition from pure JavaScript logic to interacting with the web page (Document Object Model).

## Key Concepts
- **Selecting Elements**: `document.getElementById`, `querySelector`, `querySelectorAll`.
- **Modifying Styles**: `element.style.property`, `classList.add/remove`.
- **Updating Content**: `innerText`, `innerHTML`.
- **Event Listeners**: `addEventListener('click', ...)`.

## Exercises
- **Color Switcher**: Changing background colors dynamically.
- **Font Resizer**: Adjusting text size with buttons.
- **Dark Mode Toggle**: Toggling CSS classes for a theme switch.

## How to use
1. Open `index.html` in your browser.
2. Follow the instructions in `script.js` to add interactivity.
3. Use `style.css` to define the initial looks.
