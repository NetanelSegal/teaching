# Character Operations Exercises

This folder contains exercises for working with characters (`char`) and their ASCII values in C and Java.

## Exercises
1. **Next Character**: Input a character and display the next character in the ASCII table.
2. **Digit Conversion**: Input a character representing a digit ('0'-'9'). Calculate its numerical value and display the result of multiplying that value by 10 and by 2.
3. **Digit Sum**: Input two characters, each representing a digit. Display the sum of their numerical values.
   - **Example**: Input '5' and '9' -> Output: 14.
4. **Case Conversion**: Input an uppercase letter and display the corresponding lowercase letter.
