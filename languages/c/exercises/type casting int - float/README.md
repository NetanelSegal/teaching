# Type Casting (Int / Float) 🔄

## Goal

Understand precision, division, and data type conversions.

## Instructions

Write programs to solve the following:

1. **Exact Average**: Read two integers and calculate their exact average as a decimal number (float/double). Ensure no integer division truncation occurs.
2. **Price Calculation**: Read the price of an item (float/double) and an integer quantity. Print the total cost.
3. **Rounding**: Read a floating-point number and round it to the nearest whole integer using casting (do not use Math.round).
4. **Fractional Part**: Given a decimal number, extract and print only its fractional part. (e.g., 5.75 -> 0.75).
