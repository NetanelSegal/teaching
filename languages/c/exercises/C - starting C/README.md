# Getting Started with C

## Subject: Basic I/O (printf and scanf)

1. **Print Formatted Name**
   Write a program that prints your name in this format:
   ```
   ******************
   *** Your name ***
   ******************
   ```

2. **Print Favorite Number**
   Write a program that uses `printf()` to display your favorite number.

3. **Input and Output Age**
   Write a program that asks the user for their age and uses `printf()` to display it.

## Subject: Variables and Memory Management

4. **Print Variable Sizes**
   Declare variables of type `int` and `float`, and print their sizes using `sizeof()`.

5. **Assign and Print Values**
   After printing their sizes (from the previous exercise), assign values to the variables and print their values.

## Subject: Numerical Operations and Calculations

6. **Sum of Two Numbers**
   Write a program that calculates and prints the sum of two numbers provided by the user.

7. **Product of Two Numbers**
   Write a program that calculates the product of two numbers provided by the user.
