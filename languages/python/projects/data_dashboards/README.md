# Data Dashboards Track: Insights to Deployment

Learn how to turn raw data into interactive, web-based dashboards. This track covers the full pipeline from data cleaning to visualization and deployment.

## Prerequisites
- **Data Collections:** Mastering Lists, Dictionaries, and List Comprehensions.
- **File I/O:** Reading and writing CSV and JSON files.
- **Mathematics:** Basic understanding of statistics (mean, median, trends).
- **Functions & Modules:** Organizing data processing pipelines.

## Core Libraries
- `pandas`: The "Excel of Python" for data manipulation and analysis.
- `plotly`: For creating beautiful, interactive charts and graphs.
- `streamlit`: To turn Python scripts into shareable web apps in minutes.
- `numpy`: For efficient numerical computations.

## Projects Roadmap

### Project 1: Personal Finance Tracker (Data Analysis)
Analyze your own (or mock) spending habits. Clean a CSV export from a bank and generate summary statistics.
- **Focus:** Pandas DataFrames, filtering, grouping, and basic Plotly charts.

### Project 2: Interactive Global Trends Dashboard (Visualization)
Use a public dataset (e.g., COVID-19, Climate Change, or Spotify charts) to create a multi-page interactive dashboard.
- **Focus:** Streamlit layout, user input widgets (sliders, dropdowns), and advanced Plotly visualizations (maps, heatmaps).

### Project 3: Real-time Stock or Crypto Monitor (Integration)
Build a dashboard that fetches live data from an API and updates charts in real-time.
- **Focus:** API integration (`requests`), real-time data streaming, and Streamlit caching for performance.
