# AI and Bots Track: Intelligent Agents & APIs

Explore the world of artificial intelligence by building bots that interact with users and models that learn from data.

## Prerequisites
- **JSON & APIs:** Understanding how to send and receive data via REST APIs.
- **AsyncIO:** Essential for building responsive Discord or Telegram bots.
- **OOP:** Using classes to maintain bot state and build clean ML pipelines.
- **Environment Management:** Handling API keys and `.env` files securely.

## Core Libraries
- `openai`: To interact with GPT models and DALL-E.
- `discord.py` / `python-telegram-bot`: For building chat-based agents.
- `scikit-learn`: The entry point for Machine Learning (Regression, Classification).
- `python-dotenv`: For managing secrets.

## Projects Roadmap

### Project 1: AI Chatbot with History (API Consumption)
Create a terminal-based chatbot powered by OpenAI that remembers the conversation context.
- **Focus:** API calls, message history management, and system prompting.

### Project 2: Multi-Tool Discord Bot (Integration & Async)
Build a Discord bot that can perform tasks like moderating a channel, fetching weather data, and summarizing long messages using AI.
- **Focus:** Event-driven programming, asynchronous tasks, and command handling.

### Project 3: Predictive Analytics Tool (Simple ML)
Use Scikit-Learn to build a model that predicts something useful, like house prices or movie ratings based on features.
- **Focus:** Data splitting, model training, evaluation metrics, and making predictions.
