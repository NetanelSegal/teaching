# Professional Python Software Engineering Curriculum

This is a comprehensive, job-aligned Python curriculum designed for ORT. It transitions students from fundamental programming logic to professional software engineering and data science.

## 📂 Repository Structure

-   **[lessons/](./lessons/)**: Theoretical modules following the "Hook-Theory-Lab" model. Now includes the **Professional Data Science Track (23-26)**.
-   **[exercises/](./exercises/)**: Tiered practice modules (Foundation, Logic, Algorithmic Thinking, and Industry Readiness).
-   **[projects/](./projects/)**: Domain-specific project tracks (Games, AI, Automation, and Data Dashboards).
-   **[solutions/](./solutions/)**: Reference implementations for instructors.

## 🚀 The Learning Path

1.  **Core Fundamentals (01-06):** Master variables, logic, loops, and functions.
2.  **Data Structures & Memory (07-13):** Understand how Python stores data and manages memory under the hood.
3.  **Engineering Standards (14-22):** Professional OOP, Error Handling, File Systems, and Testing with Pytest.
4.  **Data Science & Analysis (23-26):** High-performance computing with NumPy, Pandas, Visualization, and SQL.

## 🛠️ Standards & Conventions
-   **Algorithmic Thinking:** We prioritize understanding *how* things work (B-Trees, SIMD, C-Backends) over just memorizing syntax.
-   **Modern Python:** All materials use Python 3.10+ features (Pattern Matching, Type Hinting, f-strings).
-   **Job Alignment:** Exercises and projects are mapped to `roadmap.sh` and industry-standard technical interviews.
