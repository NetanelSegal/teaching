# Python Curriculum: Lessons

This directory contains the core theoretical modules of the Python Software Engineering track. The curriculum is designed to take a student from zero to a professional level, with a heavy emphasis on **Algorithmic Thinking** and **System Architecture**.

## Syllabus Overview

### Part 1: Foundations (The Basics)
- **01-03:** Variables, Logic, and Control Flow (Loops).
- **04-05:** Core Sequences (Strings, Lists, and Matrices).
- **06:** Functions and Scoping.

### Part 2: Intermediate Data & Logic
- **07-09:** Project Structure, Dictionaries, Sets, and Tuples.
- **10:** Memory Management & Object Identity (Interning, `is` vs `==`).
- **11-13:** Modern Patterns (Matching, Comprehensions, Iterators).

### Part 3: Professional Engineering
- **14-16:** Error Handling and Object-Oriented Programming (OOP).
- **17-18:** Systems (File I/O, Decorators, Context Managers).
- **19-20:** Quality & Tooling (Pytest, Type Hinting).
- **21-22:** Advanced Systems & Capstones.

### Part 4: Data Science & Professional Track (New)
- **23: NumPy & Numerical Computing** (The C-Backend & SIMD).
- **24: Pandas & Data Analysis** (Columnar Storage & Alignment).
- **25: Data Visualization** (The Artist Hierarchy & Storytelling).
- **26: SQL & Databases** (B-Trees & Query Optimization).

## How to use this
Each folder contains a `lesson.md` file. Instructors should follow the "Hook -> Theory -> Code" model provided in each document.
