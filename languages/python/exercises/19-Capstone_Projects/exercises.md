# Capstone Projects: From 0 to 100

These projects are designed to combine all the skills you've learned. Choose one or more to complete!

## Project 1: The CLI Budget Tracker (Intermediate)
**Goal:** Create a CLI tool to manage your personal expenses.
- **Features:** Add an expense (amount, category, description), view total spending, and filter by category.
- **Skills used:** Dictionaries, Lists, Functions, Exception Handling, File I/O (save/load from JSON).

## Project 2: The Quiz Master (Intermediate/Advanced)
**Goal:** Build a robust quiz application that reads questions from a file.
- **Features:** Randomize questions, track user score, provide hints, and save high scores to a leaderboard.
- **Skills used:** OOP (Question/Quiz classes), `pathlib`, `random` module, `json` (for high scores).

## Project 3: Library Management System (Advanced)
**Goal:** A full-featured library system to manage books, authors, and users.
- **Features:** Rent/Return books, search by title/author, and prevent renting a book that is already out.
- **Skills used:** Advanced OOP (Composition, Inheritance), Dataclasses, Custom Exceptions, List/Dict Comprehensions.

## Project 4: Weather API Dashboard (Advanced - Job Ready)
**Goal:** A dashboard that fetches real-time weather data from an API.
- **Features:** Fetch data for multiple cities, handle network errors gracefully, and display a formatted summary.
- **Skills used:** HTTP requests (`requests` or `httpx`), AsyncIO (optional but recommended), JSON parsing, Exception handling.

## Project 5: The Unit Testing Challenge (Professional)
**Goal:** Take one of your previous projects (like the Budget Tracker) and write a full test suite for it.
- **Features:** 100% code coverage for the core logic, using fixtures and mock objects.
- **Skills used:** `pytest`, Mocking, Fixtures, Parameterization.
