# Python Curriculum: Exercises

This directory contains the hands-on practice modules corresponding to each lesson. To build deep mastery, every module follows a **Tiered Difficulty System**.

## The Tiered Difficulty System

### 🟢 Level 1: Foundation (Mechanics)
Focused on syntax and "getting it to work." These exercises ensure the student understands the basic usage of the library or concept.

### 🟡 Level 2: Logic & Mechanics (Integration)
Focused on combining multiple concepts. This is where students start to solve small problems and handle standard data manipulation.

### 🔴 Level 3: Algorithmic Thinking (The "Engineer" Level)
The core of our curriculum. These exercises force students to think about efficiency, memory, and complex logic. They are designed to be difficult and require careful planning.

### 💼 Level 4: Industry Readiness (Job-Ready Challenges)
Added to professional modules (23-26). These simulate real-world tasks, such as cleaning financial data, optimizing database performance, or building professional visualizations.

---

## Instructions for Students
1.  Complete the `Level 1` and `Level 2` exercises for every lesson.
2.  Attempt `Level 3` even if it is difficult; this is where the most learning happens.
3.  For Data Science modules (23+), ensure you complete the `Industry Readiness` section to build your portfolio.
