# Python Curriculum: Solutions

This directory contains official reference solutions for the curriculum exercises. 

## ⚠️ Warning for Students
Looking at solutions before attempting the problem significantly slows down your learning. You should only use these files:
1.  **After** you have successfully solved the problem, to see a different or more optimized approach.
2.  **After** you have spent at least 30 minutes stuck on a single logic gate in a `Level 3` exercise.

## Structure
-   Solutions are organized by lesson number.
-   Many files contain multiple versions of a solution (e.g., "Naive" vs "Optimized").
-   **Legacy:** Contains older solutions that may follow outdated syntax or structures.
