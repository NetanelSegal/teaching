שימוש במאפיינים מותאמים אישית (Using CSS custom properties)

מאפיינים מותאמים אישית (Custom Properties), הידועים גם בשם משתני CSS, הם ישויות המוגדרות על ידי כותבי ה-CSS המכילות ערכים ספציפיים לשימוש חוזר בכל המסמך.

למה להשתמש? משתני CSS מפחיתים חזרתיות, מקלים על תחזוקה ומאפשרים שינויים דינמיים בזמן אמת (למשל החלפת ערכות נושא).

מושגים בסיסיים (Basic Concepts)

משתני CSS מוגדרים באמצעות קידומת של שני מקפים (--) וניגשים אליהם באמצעות פונקציית ה-var().

הצהרה ושימוש

:root {
  --main-bg-color: brown;
}

body {
  background-color: var(--main-bg-color);
}

טווח והורשה (Scoping and Inheritance)

משתני CSS כפופים לכללי המדרג (Cascade) וההורשה של CSS.

  • טווח גלובלי: הגדרת משתנים ב-:root הופכת אותם לזמינים בכל המסמך.
  • טווח מקומי: הגדרה בתוך סלקטור ספציפי מגבילה את המשתנה לאלמנט וצאצאיו בלבד.

ערכי ברירת מחדל (Fallback Values)

ניתן להגדיר ערך חלופי למקרה שהמשתנה אינו מוגדר:

color: var(--my-var, red); /* יציג אדום אם --my-var לא קיים */

דוגמה חיה: החלפת ערכת נושא (Theme Switching)

מכיוון שמשתני CSS הם "חיים", ניתן לשנות אותם בקלות באמצעות הוספת קלאס או דרך JavaScript.

כרטיס אינטראקטיבי

לחצו על הכפתור למטה כדי להחליף בין מצב יום למצב לילה.

:root { --bg: white; --text: black; }
.dark-mode { --bg: #222; --text: #eee; }

body {
  background-color: var(--bg);
  color: var(--text);
}
הערה: שמות המשתנים רגישים לאותיות גדולות/קטנות (Case-sensitive).
