אנימציות ב-CSS (Using CSS Animations)

אנימציות ב-CSS מאפשרות להנפיש מעברים בין תצורות סגנון שונות. האנימציה מורכבת משני חלקים: תיאור האנימציה (Animation Configuration) וסט של נקודות מפתח (Keyframes) המציינות את תחילת וסיום מצבי הסגנון של האנימציה, וכן נקודות ציון אפשריות בדרך.

יתרונות: קלות לשימוש עבור אנימציות פשוטות, רצות היטב גם תחת עומס מערכת, ומאפשרות לדפדפן לשלוט ברצף האנימציה ולייעל את הביצועים.

קביעת תצורת האנימציה (The Animation Configuration)

כדי ליצור רצף אנימציה, עלינו להשתמש בתכונות ה-animation. אלו מאפשרות לקבוע את התזמון, משך הזמן ופרטים נוספים.

  • animation-name: השם של ה-Keyframes שיופעלו.
  • animation-duration: משך הזמן של מחזור אנימציה אחד.
  • animation-timing-function: עקומת התאוצה (למשל ease, linear).
  • animation-delay: השהיה לפני תחילת האנימציה.
  • animation-iteration-count: מספר הפעמים שהאנימציה תחזור על עצמה.
  • animation-direction: האם האנימציה תתנגן קדימה, אחורה או לסירוגין.

דוגמה בסיסית:

Slide
.box {
  animation-name: slide;
  animation-duration: 2s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes slide {
  from { transform: translateX(100px); }
  to { transform: translateX(-100px); }
}

הגדרת רצף האנימציה באמצעות @keyframes

הפקודה @keyframes מגדירה את המראה של האלמנט בנקודות זמן שונות במהלך האנימציה.

ניתן להשתמש במילות המפתח from ו-to (השקולות ל-0% ו-100%), או באחוזים מדויקים כדי להגדיר שלבים בדרך.

ביצועים: מומלץ להנפיש רק תכונות שאינן גורמות ל-Reflow (חישוב מחדש של הפריסה), כמו transform ו-opacity.

דוגמה מהעולם האמיתי: Skeleton Screen

טכניקה נפוצה להצגת טעינה המדמה את מבנה התוכן העתידי.

@keyframes shimmer {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(100%); }
}

.shimmer::after {
  content: "";
  position: absolute;
  top: 0; left: 0; width: 100%; height: 100%;
  background: linear-gradient(90deg, transparent, rgba(255,255,255,0.4), transparent);
  animation: shimmer 1.5s infinite;
}

נגישות (Accessibility)

חלק מהמשתמשים עלולים לחוות אי-נוחות מאנימציות. עלינו לכבד את הגדרות המערכת שלהם:

@media (prefers-reduced-motion: reduce) {
  .animate-slide, .shimmer::after {
    animation: none;
  }
}
