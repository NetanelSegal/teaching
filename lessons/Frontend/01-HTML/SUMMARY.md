# סיכום HTML

HTML (HyperText Markup Language) היא השפה שבה אנחנו בונים את המבנה של דפי אינטרנט.

## 1. מבנה בסיסי
כל דף HTML מתחיל עם `<!DOCTYPE html>` ומכיל את התגיות הבאות:
- `<html>`: עוטפת את כל הדף.
- `<head>`: מכילה מידע על הדף (כותרת, קידוד, קישורים ל-CSS).
- `<body>`: מכילה את התוכן שרואים בדפדפן.

## 2. תגיות נפוצות
- **כותרות**: `<h1>` (הכי חשובה) עד `<h6>`.
- **פסקאות**: `<p>`.
- **קישורים**: `<a href="URL">טקסט</a>`.
- **תמונות**: `<img src="path" alt="תיאור">`.
  - **תמונות רספונסיביות**: שימוש ב-`srcset` ו-`<picture>` להצגת תמונות שונות לפי גודל המסך.
- **רשימות**: `<ul>` (לא מסודרת), `<ol>` (מסודרת), `<li>` (פריט ברשימה).

## 3. טפסים (Forms)
טפסים משמשים לקבלת קלט מהמשתמש:
- `<form>`: הקונטיינר של הטופס.
- `<input>`: שדה קלט (type="text", type="email", type="password").
- `<button>`: כפתור שליחה.
- `<label>`: תווית לשדה קלט.

## 4. סמנטיקה ונגישות (Accessibility)
שימוש בתגיות סמנטיות עוזר למנועי חיפוש ולנגישות:
- `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`.
- **ARIA Attributes**: שימוש ב-`aria-label`, `role` וכו' כדי להפוך את האתר לנגיש יותר לקוראי מסך.
